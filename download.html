jQuery Adapter

batman.js has no hard browser-land dependencies. That said, there are a number of things that are beyond the scope of the microframework, for example XHR and animation. These things are wrapped by batman objects, but the default build includes a small dependency file that outsources these things to other micro libraries.

The batman.solo.js dependency file adds less than 5kb to the framework, and is likely something you'll want to include. However, if you're already using jQuery in your app, jQuery can do all of these things for you. We've included an alternate jQuery adapter that will add no additional size to batman.js aside from the weight of jQuery itself.

If you would like to use jQuery in your project, simply source batman.jquery.js instead of batman.js, in addition to your jquery.js.
